# Kalambury Zuzanny

Romantyczna gra w kalambury dla dwóch graczy - Mateusza i Zuzanny.

## Uruchomienie

```bash
cd kalambury_zuzanny
pip install -r requirements.txt
streamlit run app.py
```

## Funkcje

- 3 poziomy trudności (łatwy, średni, trudny)
- 300 haseł po polsku
- System punktacji z bonusem czasowym
- Możliwość przelosowania hasła
- Historia wyników
- Piękny romantyczny interfejs z animacjami
